# 操作系统课程项目

##  一、项目说明

###  1. 进程管理-电梯调度（ProcessManagement-elevator）

(1) 基本任务：

某一层楼20层，有五部互联的电梯。基于线程思想，编写一个电梯调度程序。

(2) 功能描述：

- 电梯应有一些按键，如：数字键、关门键、开门键、上行键、下行键、报警键等。
- 有数码显示器指示当前电梯状态。
- 每层楼、每部电梯门口，有上行、下行按钮、数码显示。

(3) 其他要求：

- 所有电梯初始状态都在第一层。
- 每个电梯没有相应请求下，则应该在原地保持不动。
